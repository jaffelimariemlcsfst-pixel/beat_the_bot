# 🤖 BeatTheBot

> ⚔️ A Flutter quiz game where you compete against AI across multiple knowledge arenas. Answer challenges, get judged instantly, earn XP, and level up.

---

## ✨ Features

- ✍️ **Multiple question types** — Write, Multiple Choice, Draw, Pick an Image
- 🖼️ **Real images** fetched from Unsplash for pick-image questions
- 🧠 **AI-powered questions** generated by Llama 3.3
- ⚖️ **AI answer judging** with detailed feedback and optimal answers
- ⚡ **Question caching** for faster load times
- 🏆 **XP system** and leveling up
- 🔥 **Daily streaks** to keep you coming back
- ❤️ **Lives system** — 3 lives per game
- ⏱️ **60-second timer** per question
- 🗺️ **Multiple topic arenas** to choose from

## 🚀 Getting Started

### 📋 Prerequisites
- 🐦 Flutter SDK installed
- 🔑 A Groq API key from [console.groq.com](https://console.groq.com)
- 🖼️ An Unsplash API key from [unsplash.com/developers](https://unsplash.com/developers)

### 📦 Installation

**1. Clone the repo**
```bash
git clone https://github.com/jaffelimariemlcsfst-pixel

cd beat_the_bot
```

**2. Install dependencies**
```bash
flutter pub get
```

**3. Create a `.env` file in the project root** 🔐
```env
GROQ_API_KEY=your_groq_key_here
UNSPLASH_ACCESS_KEY=your_unsplash_key_here
```

**4. Run the app** 🚀
```bash
flutter run
```

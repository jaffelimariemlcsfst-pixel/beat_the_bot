# 🤖 BeatTheBot

A Flutter quiz game where you compete against AI across multiple topics.

## Features
-  Multiple question types (write, multiple choice, draw, pick image)
-  Real images fetched from Unsplash for pick-image questions
-  AI-powered questions generated by Llama 3.3
-  AI answer judging with detailed feedback
-  Question caching for faster load times
-  XP system and leveling up
-  Daily streaks
-  Lives system (3 lives per game)
-  60-second timer per question
-  Multiple topics to choose from

## Getting Started

### Prerequisites
- Flutter SDK
- Groq API key
- Unsplash API key

### Installation
1. Clone the repo
```
   git clone https://github.com/ibtihelsaidi2006/BeatTheBot-.git
```
2. Install dependencies
```
   flutter pub get
```
3. Create a `.env` file in the project root
```
   GROQ_API_KEY=your_groq_key_here
   UNSPLASH_ACCESS_KEY=your_unsplash_key_here
```
4. Run the app
```
   flutter run
```

## Built With
- Flutter & Dart
- Groq API (Llama 3.3 70B)
- Unsplash API
- Supabase
- Provider (state management)
- GoRouter (navigation)
